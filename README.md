# Tic Tac Toe game using cpp

Watch the Youtube Video from [here](https://youtu.be/vY58Z-5Fq80)

# Project setup

### Clone the project
```shell
git clone https://github.com/Code-Challenge-YT/Tic-Tac-Toe-game-using-cpp.git
```

### Compile the code
```shell
g++ main.cpp -o main.exe
```

### Run compiled output
```shell
./main.cpp
```
